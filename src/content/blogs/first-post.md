# Welcome to My Blog

This is your first demo blog post! You can edit this content and add your own blog posts.

## Getting Started

To create a new blog post, simply add a new `.md` file in the `src/content/blogs/` directory with the following frontmatter:

```yaml
---
title: Your Post Title
description: A brief description of your post
pubDate: 2026-07-14
tags: [tag1, tag2]
sheet: BLOG-XXX (optional)
draft: false
---
```

## What's Next?

- Edit this post by modifying `src/content/blogs/first-post.md`
- Create additional blog posts in the same directory
- Update your blog styling in the pages that display these posts

Happy blogging! 📝
